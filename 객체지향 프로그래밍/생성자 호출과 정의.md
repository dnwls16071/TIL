# ❗생성자란?

- 생성자란, 객체를 만드는 특별한 메소드다.

![CR0MkmA](https://github.com/dnwls16071/TIL/assets/106802375/44e2916e-cc68-4e0e-8602-568fa5fb8ea0)

# ❗생성자의 역할

- 생성자는 두 가지 역할을 한다.

  - 객체 생성 - 객체를 만드는 것
 
  - 객체 초기화 - 객체의 필드 값을 설정하는 것  

### 이전 방식

```java
/* 1. 객체 생성 */
Cat c = new Cat();
/* 2. 필드 초기화 */
c.name = "네로";
c.weight = 3.78;
c.age = 3;
```

### 생성자를 활용한 방식

```java
/* 객체 생성과 동시에 초기화 */
Cat c = new Cat("네로", 3.78, 3);
```
