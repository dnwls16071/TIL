# 단지 번호 붙이기

그림1과 같이 정사각형 모양의 지도가 있다. 1은 집이 있는 곳을, 0은 집이 없는 곳을 나타낸다.

철수는 이 지도를 가지고 연결된 집들의 모임인 단지를 정의하고, 단지에 번호를 붙이려 한다. 

여기서 연결되었다는 것은 어떤 집이 좌우, 혹은 아래위로 다른 집이 있는 경우를 말한다.

대각선상에 집이 있는 경우는 연결된 것이 아니다.

그림2는 그림1을 단지별로 번호를 붙인 것이다. 지도를 입력하여 단지수를 출력하고, 각 단지에 속하는 집의 수를 오름차순으로 정렬하여 출력하는 프로그램을 작성하시오.

## ▣ 입력설명

첫 번째줄에는 지도의 크기 N(정사각형이므로 가로와 세로의 크기는 같으며 5≤N≤25)이 입력되고 그 다음 N줄에는 각각 N개의 자료(0 혹은 1)가 입력된다

## ▣ 출력설명

첫 번째줄에는 총 단지수를 출력하시오. 그리고 각 단지내의 집의 수를 **오름차순으로 정렬**하여 한줄에 하나씩 출력하시오

## ▣ 입력예제 1

7

0110100

0110101

1110101

0000111

0100000

0111110

0111000

## ▣ 출력예제 1

7

8

9

▣ 첫 번째 풀이(BFS, 너비 우선 탐색)

```python
import sys
input = sys.stdin.readline
from collections import deque

N = int(input())
graph = [list(map(int, input().strip())) for _ in range(N)]
visited = [[False] * N for _ in range(N)]

def BFS(y, x):
    cnt = 1
    visited[y][x] = True
    queue = deque()
    queue.append((y, x))
    while queue:
        b, a = queue.popleft()
        dy = [1, 0, -1, 0]
        dx = [0, 1, 0, -1]
        for i in range(4):
            ny = b + dy[i]
            nx = a + dx[i]
            if 0 <= ny < N and 0 <= nx < N and not visited[ny][nx]:
                if graph[ny][nx] == 1:
                    visited[ny][nx] = True
                    queue.append((ny, nx))
                    cnt += 1
    return cnt
    
res = []
for i in range(N):
    for j in range(N):
        if graph[i][j] and not visited[i][j]:
            res.append(BFS(i, j))

print(len(res))
res.sort()
for i in res:
    print(i)
```
